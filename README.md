## 크롤링 방법

- search 서비스에 적절한 쿼리를 제공해서 시행년도, 시행월, 학년을 조절하여 시험 데이터를 획득한다.
- 문제별 기본 정보 + id 를 획득할 수 있고, 이는 item 서비스를 통해 문항 정보를 더 자세히 가져올 수 있다.
- 따라서 search 서비스에서 가져온 리스트를 순회하면서 한 회차 시험지를 저장하는 방식으로 진행하면 된다.

## 필요한 (사용할만한) 데이터 정의

### search 결과

- cate_nm_1: 카테고리 1
  - 예시: "국어"
- cate_nm_2: 카테고리 2
  - 예시: "화법"
- cate_nm_3: 카테고리 3
  - 예시: "화법의 원리와 실제"
- cate_nm_4: 카테고리 4
  - 예시: "발표"
- cate_nm_5: 카테고리 5
  - 예시: "발표 표현 전략 사용하기"
- point: 문항 점수
  - 예시: "2"
- year: 시행연도
  - 예시: "2021"
- title: 시험 제목
  - 예시: "고1 6월 학평(부산)"
